flex container: vùng màu vàng(thẻ chứa)

flex items: 3 thẻ con

Tất cả các giá trị màu đỏ đều có thể thay đổi

    main axis: trục chính nằm ngang
  • main start
  • main end
    cross axis: trục dọc
  • cross start
  • cross end
    Kích thước flex items:
  • main size: kích thước có chiều song song với main axis
  • cross size: kích thước có chiều song song với cross axis
